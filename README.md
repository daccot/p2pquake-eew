# P2P地震情報アラート EEWカウントダウン完成版

P2P地震情報 WebSocket API を監視し、最大震度3以上の地震情報、または緊急地震速報を受信したときに、ブラウザページ上へ警告オーバーレイを表示する Chrome Extension です。

## 追加済み機能

- EEWカウントダウン
- P波 / S波 到達予測
- ユーザー地点設定
- 震源距離推定
- 業務影響ラベル
- アイコン状態変化
  - 通常
  - 地震検知
  - EEW
- EEW時のアイコン点滅
- バッジ通知
  - 地震: `震`
  - EEW: `警`
- 通知履歴
- 地域フィルタ
- 10分 / 1時間ミュート
- 警告音
- 通常通知テスト
- EEWカウントダウンテスト
- WebSocket再接続ボタン

## 導入手順

1. ZIPを展開します。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をONにします。
4. 「パッケージ化されていない拡張機能を読み込む」を押します。
5. 展開したフォルダを選択します。

## 動作確認

1. `https://example.com/` など通常のWebページを開きます。
2. 拡張のオプション画面を開きます。
3. 「EEWカウントダウンテスト」を押します。
4. ページ上にカウントダウン付きオーバーレイが表示されます。
5. 拡張アイコンが点滅し、バッジ `警` が表示されます。

## 注意点

- `chrome://` ページでは表示できません。
- Chrome Web Store のページでは表示できません。
- PDFビューアなど、一部特殊ページでは content script が動作しません。
- 到達予測は簡易推定です。公式な防災判断には気象庁・自治体などの情報を優先してください。
- WebSocketは Chrome 116+ の Manifest V3 Service Worker 前提です。
